Automator

„Automator“  allows users without any programming knowledge to create basic computer automations.
You can import and export the automation. Undo/Redo operations and change the order of the automation with the Up/Down button or Drag and Drop option.
The Functionality allows to :
  • click on images/buttons
  • insert text
  • search and copy a word
  • paste text
